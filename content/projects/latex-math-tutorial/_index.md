summary: Write and render LaTeX math expressions natively inside your Markdown content using KaTeX.

---

Mathematical notations can be written directly in your content using LaTeX syntax, which renders beautifully:

Inline math example: `$E = mc^2$`.

Block math example:

```latex
\[
\int_0^\infty e^{-x} \, dx = 1
\]
```

This rendering uses KaTeX for quick, reliable math typography without external images or scripts.
