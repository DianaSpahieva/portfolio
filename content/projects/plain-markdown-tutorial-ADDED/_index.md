Markdown is a lightweight markup language that lets you write using an easy-to-read, easy-to-write plain text format.

- Use headings, lists, links, images, code blocks, and more using simple symbols.
- Compatible with all text editors and version control.
- Your Markdown files are fully portable, usable outside this platform indefinitely.

Here is a sample snippet:

```markdown
# Hello World
This is a paragraph with **bold** and *italic* text.
```

Learn more about Markdown at [https://commonmark.org](https://commonmark.org).
